# Project Proposal: Conversation to Instagram Post Assistant

## Problem Statement
People often come up with useful ideas and insights during everyday conversations. However, converting these ideas into Instagram posts requires extra time and effort, so many original thoughts are never shared.

## Proposed Solution
This project proposes a simple web-based application that allows a user to upload or record short conversational audio. The system converts the audio into text and suggests 1–3 Instagram-style post captions based on the conversation. The system only provides suggestions and does not automatically post content.

## Scope

### In Scope
- Web-based prototype
- Upload or record short audio clips
- Speech-to-text conversion using an existing model
- Generate Instagram-style caption suggestions
- Display transcript and suggested captions

### Out of Scope
- Automatic posting to Instagram
- Trend analysis or engagement optimization
- Long-duration audio recording
- Speaker identification or diarization
- Image generation

## Technologies
- Python
- Streamlit

## Technology Considerations
This project will rely on existing speech-to-text and text generation technologies. Specific models and tools will be selected after a formal technology evaluation.


## Evaluation Plan
The system will be evaluated by comparing raw transcript text with AI-generated caption suggestions. User feedback will be collected to determine which output is clearer and more suitable as an Instagram post.
